# Packly Developer Preview v0.59.1

This follow-up Developer Preview release adds the redacted doctor bundle path for safer install and MCP connection debugging.

## Included

- `packly` CLI
- `packly-mcp` local stdio MCP server
- `packly doctor --bundle`
- MCP context routing tools for Pack find/brief/activate/read/session history
- MCP install/update planning and approval-request tools
- Claude Code, Codex, and Cursor MCP config helpers
- Homebrew tap: `brew tap Orvek-dev/packly && brew install packly`

## Artifacts

- `packly-developer-preview-v0.59.1-aarch64-apple-darwin.tar.gz`
- `packly-developer-preview-v0.59.1-x86_64-unknown-linux-gnu.tar.gz`
- `packly-developer-preview-v0.59.1-x86_64-pc-windows-msvc.zip`
- `checksums.txt`

## Integrity

- `v0.59.1` is the only supported public Developer Preview release.
- The shell installer pins the `checksums.txt` SHA-256 digest for the default release before verifying archives.
- Build-host path remapping is applied to release binaries to avoid exposing local user or runner home paths.
- Detached signatures, Sigstore attestations, SBOMs, macOS Developer ID notarization, and Windows code signing are not yet included.

## Validation

- `node scripts/packly/check-version-consistency.mjs`
- `cargo check -p packly -p packly-mcp`
- `node apps/desktop/scripts/prepare-sidecar.mjs && cargo check --manifest-path apps/desktop/src-tauri/Cargo.toml`
- `cargo fmt --all -- --check`
- `cargo test --workspace`
- `python3 scripts/packly/mcp_smoke.py --packly-bin target/debug/packly --mcp-bin target/debug/packly-mcp`
- `python3 scripts/packly/mcp_client_config_smoke.py --packly-bin target/debug/packly --mcp-bin target/debug/packly-mcp`
- `packly --json doctor --no-workspace --mcp-bin ./target/debug/packly-mcp --bundle`

## Known Limitations

- Developer Preview, not stable Packly.
- Public repo contains docs, sample Packs, issue templates, installer, and release artifacts only.
- Source code remains private during this preview.
- macOS Intel and Linux ARM64 binaries are not published yet.
